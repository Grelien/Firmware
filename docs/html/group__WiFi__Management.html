void wifi_init_softap ( void  )

Initialize and start WiFi Access Point.

Initialize WiFi in Access Point mode.

Configures the ESP32 as a WiFi Access Point with predefined network settings. Creates a standalone WiFi network that other devices can connect to for accessing the web interface. Handles network interface initialization, WiFi configuration, and AP startup.
