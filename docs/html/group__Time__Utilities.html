Format current system time for display.

Get current time as formatted string.

Retrieves the current system time and formats it into a human-readable string suitable for display in the web interface and system logs. Uses strftime() to create a standardized format.

Parameters
bufferOutput buffer to store the formatted time string
buffer_sizeSize of the output buffer in bytes
Note
Format: "YYYY-MM-DD HH:MM:SS Day" Example: "2025-07-08 14:30:25 Monday"
If system time is not set, may show epoch time (1970-01-01)
