Detailed Description

WiFi Access Point Configuration and Management.

Author
Motor Controller Team
Date
2025

This module implements WiFi Access Point functionality for the motor controller. It creates a standalone WiFi network that allows devices to connect directly to the ESP32 without requiring an existing WiFi infrastructure. The AP mode enables web interface access for motor control and configuration.
