Turn the motor OFF.

Turn the motor OFF.

Deactivates the motor by:

  1. Setting pins to floating state for safety
  2. Updating global motor state
  3. Logging the action

Uses floating pins instead of HIGH signal for additional safety.

◆ set_pins_to_float()

void set_pins_to_float ( void  )

Set motor control pins to floating state.

This function configures the motor control pin as input with floating state, effectively disconnecting the motor. This is used for safety when the motor should be OFF.

Set motor control pins to floating state.

Configures the motor control pin as input with floating state, effectively disconnecting the motor control signal. This ensures the motor is safely OFF and not consuming power.
