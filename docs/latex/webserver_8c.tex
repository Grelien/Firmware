\doxysubsection{Detailed Description}
HTTP Web Server Implementation for Motor Controller. 

\begin{DoxyAuthor}{Author}
Motor Controller Team 
\end{DoxyAuthor}
\begin{DoxyDate}{Date}
2025
\end{DoxyDate}
This module implements a comprehensive HTTP web server that provides a web-\/based interface for motor control and management. It serves HTML pages, handles REST API endpoints, and manages all user interactions through the web interface. The server supports motor control, schedule management, device settings, and system monitoring.
