A comprehensive motor control system built on ESP32 with Wi\+Fi Access Point mode and web-\/based interface. The system allows remote control of a motor with scheduling capabilities, real-\/time monitoring, and persistent storage.\hypertarget{md_README_autotoc_md1}{}\doxysubsection{Features}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item {\bfseries{Wi\+Fi Access Point Mode}}\+: Creates its own Wi\+Fi network for direct connection
\item {\bfseries{Web Interface}}\+: Modern, responsive web UI for motor control and configuration
\item {\bfseries{Motor Scheduling}}\+: Add, edit, and delete motor schedules with time-\/based automation
\item {\bfseries{Real-\/time Status}}\+: Live motor status, system uptime, and schedule monitoring
\item {\bfseries{Persistent Storage}}\+: Settings and schedules saved to flash memory using NVS
\item {\bfseries{History Tracking}}\+: Maintains history of motor operations and schedule executions
\item {\bfseries{Time Synchronization}}\+: SNTP time synchronization with timezone support (GMT+5\+:30 Sri Lanka)
\item {\bfseries{Device Configuration}}\+: Customizable device name and location settings
